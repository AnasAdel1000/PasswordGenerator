# 🔑 Simple Password Generator

This is my very first Android app! A simple, secure, and **offline** tool to generate random passwords.

## 🚀 Features
* **100% Offline:** No data leaves your device.
* **Simple UI:** Easy to use and fast.
* **Customizable:** you can use other secure methods in code.
